第1回 WWW と HTML

HTML文書の構成が習いました。そして、各要素に使い方法を勉強しました。<head>の中身はページに表示しません。 <title>はタブに表示する文字を入力します。<p>は正文を入力します。<hx>は見出しと段落に使うための、xにある自然数が小さいほど、文字サイズが大きく、太字で表示されます。 <ul><ol>はそれぞれ番号なしのリストを作る要素と番号付きのリストを作る要素です、二つともリストの1つの項目は<li> 要素として記述します。 あと画像の埋め込みための<img>、srcは画像ファイル、altは画像が表示されない時の補充説明。

演習の成果:演習1-4: HTMLのさまざまな要素

第2回CSS による Web ページの装飾

今回の授業では、HTMLで記述されたWebページに対して、CSSを用いて見た目を装飾する方法を学びます。 文字の色やサイズ、配置、背景色、枠線の設定など、 また、外部スタイルシート、内部スタイルシート、の違いや使い分けについても理解できました。

演習の成果:演習2-3: CSS の編集
