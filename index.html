数当てゲーム

このゲームでは、HTML と JavaScript を使って作成し、 1〜10の中からランダムに選ばれた整数を当てるシンプルな数当てゲームです。 予想値を4に固定して、予想回数は最大3回で、正解やヒント、ゲーム終了のメッセージが表示されます。 HTMLではテキストボックスとボタンを用意し、JavaScriptでイベント処理とDOM操作を使って判定結果（正解・不正解）を <p id="result"> に表示します。

各授業回の学習事項

第1回 WWW と HTML

今回の授業では、HTMLの基本構造と要素について学びます。 Webページの作成のためのHTML文書の書き方、要素の意味や使い方、属性の指定方法、 特殊文字の扱いなどを理解し、実際にコードを書いてWebページを作成できるようになることを目指します。

演習の成果:演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回の授業では、HTMLで記述されたWebページに対して、CSSを用いて見た目を装飾する方法を学びます。 文字の色やサイズ、配置、背景色、枠線の設定など、 また、外部スタイルシート、内部スタイルシートの違いや使い分けについても理解できました。

演習の成果:演習2-3: CSS の編集

第3回 JavaScript の基本

今回の授業では、JavaScript の基本構文やエラー処理、オブジェクトの使い方、デバッグ方法を学びます。 Java との違いを比較しながら、変数の扱いや関数の定義、配列やオブジェクトの操作方法について実践的に理解を深めます。 また、Firefox の開発ツールを用いたエラー確認やブレークポイントによるデバッグ技法を体験しました。

演習の成果:演習3-6: オブジェクトの参照

第4回 イベントの処理と DOM

この授業では、HTMLのフォーム要素とJavaScriptによるイベント処理、DOM操作の基本を学びます。 具体的には、tashizan.htmlとtashizan.jsを用い、2つの数値を入力して「計算」ボタンを押すと、 足し算の結果がページ上に表示されるような仕組みを実装します。これにより、ユーザー操作に応じた動的なWebページの作成方法を習得します。

演習の成果:演習4-10: 足し算ページ/a>
